**Polymer** is a protocol that connects Ethereum's Layer 2 (L2) ecosystem through the **Inter-Blockchain Communication (IBC) protocol**. As the first implementation of virtual IBC on Ethereum, Polymer enables seamless interoperability between different blockchain networks.

## Why Polymer?

- Latency: 2-3 seconds (Near block time)
- Security: Block headers sourced from Sequencer preconfs
- Rollup Integration: Add support for new rollups within minutes
